# Assets Folder

This folder should contain the following image assets for the StrayConnected website:

## Required Images:

1. **logo.png** (40x40px or higher)
   - The StrayConnected logo for the navigation bar
   - Should be a square image with transparent background

2. **app-icon.png** (512x512px recommended)
   - The application icon shown in the download section
   - Should be a high-quality square icon

3. **app-mockup.png** (800x1600px or similar phone aspect ratio)
   - A mockup image of the app showing on a phone screen
   - Can be created using tools like:
     - Figma
     - Canva
     - MockuPhone
     - Smartmockups

## Image Guidelines:

- Use PNG format for logos and icons (supports transparency)
- Use JPG or PNG for mockups
- Optimize images for web (compress without losing quality)
- Use modern formats like WebP for better performance (optional)

## Tools for Creating Mockups:

- **Figma**: https://figma.com (Free design tool)
- **Canva**: https://canva.com (Easy mockup templates)
- **Shots**: https://shots.so (Beautiful mockup generator)
- **MockuPhone**: https://mockuphone.com (Free device mockups)

## Placeholder Images:

If you don't have images yet, you can use:
- https://placeholder.com for temporary placeholders
- Create simple colored squares with your brand colors
- Use icon generators like https://favicon.io
